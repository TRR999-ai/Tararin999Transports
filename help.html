User Portal — หน้าหลัก
ศูนย์กลางนำทางไปทุกระบบ พร้อมระบบล็อกอิน
🔐 ล็อกอินครั้งแรก (ตั้งรหัสผ่าน)
  1. 1เปิดไฟล์ project4/index.html — จะขึ้นหน้า "ตั้งรหัสผ่านครั้งแรก"
  2. 2พิมพ์ รหัสผ่าน ที่ต้องการ (ขั้นต่ำ 4 ตัวอักษร) ในช่องแรก
  3. 3พิมพ์ยืนยันรหัสผ่านอีกครั้ง → กด ตั้งรหัสผ่าน
  4. 4เข้าสู่ระบบได้ทันที เครื่องนี้จะจำ session ไว้ไม่ต้องล็อกอินซ้ำ
🔑 ล็อกอินปกติ
  1. 1เปิดหน้า portal — ถ้า session หมด จะขึ้นหน้าล็อกอิน
  2. 2พิมพ์รหัสผ่าน → กด เข้าสู่ระบบ หรือกด Enter
  3. 3ถ้าเปิดหน้าระบบย่อยโดยตรง (เช่น fleet_view) จะ redirect มาล็อกอินที่นี่ก่อน แล้วกลับไปหน้าเดิมอัตโนมัติ
💡เครื่องเดิม session อยู่ตลอด — ไม่ต้อง login ซ้ำทุกวัน ยกเว้นมีการเปลี่ยนรหัสผ่าน
🔄 เปลี่ยนรหัสผ่าน
  1. 1คลิกที่ 👤 TARARIN 999 มุมขวาบน
  2. 2ใส่ รหัสผู้ดูแล (admin code) ก่อน — ผู้ดูแลระบบเท่านั้นรู้รหัสนี้
  3. 3พิมพ์รหัสผ่านใหม่ + ยืนยัน → กด เปลี่ยนรหัสผ่าน
  4. 4ระบบ logout อัตโนมัติ — ทุกเครื่องต้อง login ใหม่ด้วยรหัสใหม่
ℹ️ปุ่ม 🚪 ออกจากระบบ มุมขวาบน — ล้าง session ของเครื่องนั้นทันที

Fleet Dashboard — ระบบติดตามกำลังรถ
ดึงข้อมูลจาก Google Sheets แบบ real-time วิเคราะห์ต้นทุน + สถานะรถ
📊 ภาพรวม KPI
  1. 1หน้าแรกแสดง KPI รวม — จำนวน trips, km รวม, ค่าน้ำมันรวม, ฿/km เฉลี่ย
  2. 2กด dropdown เดือน มุมขวาบนเพื่อเปลี่ยนเดือนดู
  3. 3กราฟแท่งแสดงยอดรายเดือน — hover เพื่อดูตัวเลขละเอียด
🚦 กำลังรถวันนี้ (Board)
  1. 1แสดงการ์ดรถทุกคัน พร้อมสถานะ ทะเบียน คนขับ และงานล่าสุด
  2. 2กดการ์ด เพื่อดูรายละเอียด — ตั้งสถานะ ✅ พร้อม ❌ ไม่พร้อม 🏖️ ลา
  3. 3แถบด้านบน filter ตาม ประเภทรถ — กดชิปเพื่อกรองเฉพาะกลุ่ม
  4. 4ปุ่ม ⚙ จัดการทะเบียน — เรียงลำดับ/ซ่อนรถ และสร้างประเภทรถเอง
💡สถานะ "พร้อม/ไม่พร้อม" รีเซ็ตอัตโนมัติทุกเที่ยงคืน — "ลา" ไม่รีเซ็ต
🔍 งานล่าสุดแต่ละคัน
  1. 1แสดงการ์ดรถพร้อมงานล่าสุด — วันนี้ เมื่อวาน บอกความใหม่
  2. 2ช่อง Note ในการ์ด — พิมพ์หมายเหตุ บันทึกอัตโนมัติ
  3. 3กด ทะเบียน ในการ์ด → ไปหน้าสรุปรายคันของรถนั้นทันที
  4. 4filter แถบบน: ตาม สถานะ หรือ ประเภทรถ
🚗 สรุปรายคัน + 👤 สรุปรายคนขับ
  1. 1ตารางสรุป trips, km, ค่าน้ำมัน, ฿/km แยกตามทะเบียน/คนขับ
  2. 2กดหัวคอลัมน์เพื่อ sort — กดซ้ำเพื่อกลับทิศ
  3. 3กดแถวรถ/คนขับ เพื่อขยายดู ประวัติ 30 งานล่าสุด
📈 วิเคราะห์
  1. 1เปรียบเทียบเดือนนี้ vs เดือนก่อน — KPI 4 ตัว พร้อม % เปลี่ยนแปลง
  2. 2Top Routes — chart + ตาราง 12 เส้นทางต้นทุนสูงสุด
  3. 3Trend ฿/km รายคัน — เลือกทะเบียนจาก dropdown → กราฟ trend รายเดือน
🏦 Statement วิเคราะห์
  1. 1วิธีที่ 1 (อัปโหลดไฟล์): กด 📂 อัปโหลดไฟล์ → เลือกไฟล์ .xlsx จาก TMB Transaction Report
  2. 2วิธีที่ 2 (Google Sheets): ตั้งค่า Sheet ID ใน Settings → กด ☁️ ดึงจาก Sheets → เลือกเดือน
  3. 3ดู 4 tab: 🚗 รายคัน / 📍 สถานีน้ำมัน / 🗺️ รายจังหวัด / 📅 รายวัน
  4. 4คลิก ที่แถวใดก็ได้ใน 3 tab แรกเพื่อ drill-down ดูรายละเอียด
  5. 5กด bar ใน tab รายวัน → ดูว่าวันนั้นรถไหนเติมที่ไหนบ้าง
💡ทะเบียนที่มี ⚠️ หมายถึงไม่พบใน Trip Sheet — km/ลิตร คำนวณไม่ได้ ตรวจสอบรูปแบบชื่อทะเบียน

Fleet Card Batch Update — อัปเดตวงเงินบัตรน้ำมัน
อัปเดตวงเงินบัตรน้ำมันหลายใบพร้อมกัน export ไฟล์ส่ง TMB
📋 ขั้นตอนการใช้งาน
  1. 1ระบบโหลดข้อมูลบัตรจาก Google Sheet อัตโนมัติ (หมายเลขบัตร + ชื่อ + วงเงินปัจจุบัน)
  2. 2แก้ไข วงเงิน ในตารางโดยตรง — คลิกที่ช่องวงเงินแล้วพิมพ์ค่าใหม่
  3. 3หรือกด 📋 วางวงเงิน — วาง list วงเงินทีละบรรทัด (ต้องตรงลำดับกับตาราง)
  4. 4กด ➕ เพิ่มบัตร เพื่อเพิ่มบัตรใหม่ทีละใบ หรือวางหลายใบพร้อมกัน
  5. 5เมื่อแก้ครบ กด Export → ดาวน์โหลดไฟล์ Excel พร้อมส่งธนาคาร
ℹ️ไฟล์ที่ export จะใช้ Template TTB อัตโนมัติถ้าโหลดได้ — ถ้าไม่มี template จะ export แบบ preview
🗑️ ลบบัตร
  1. 1ติ๊ก checkbox หน้าแถวที่ต้องการลบ
  2. 2กด 🗑 ลบที่เลือก

คำนวณต้นทุนน้ำมันตามระยะทาง + จัดการตารางระยะทาง
🧮 คำนวณต้นทุนน้ำมัน
  1. 1เลือกเมนู 🧮 คำนวณต้นทุน จาก sidebar
  2. 2เลือก ต้นทาง → ปลายทาง จาก dropdown
  3. 3ใส่ ราคาน้ำมัน (฿/ลิตร) และ อัตราสิ้นเปลือง (กม/ลิตร)
  4. 4กด คำนวณ → ระบบแสดงต้นทุนน้ำมัน ไป-กลับ พร้อม breakdown
  5. 5ผลการคำนวณบันทึกใน แดชบอร์ด อัตโนมัติ
📊 ตารางระยะทาง
  1. 1หน้าแรก — แสดง matrix ระยะทางระหว่างทุกสถานที่
  2. 2คลิกที่ช่องใน matrix เพื่อดูรายละเอียดเส้นทาง
  3. 3กด 🏠 แดชบอร์ด → ดูประวัติการคำนวณล่าสุด + ตั้งค่าน้ำมันปัจจุบัน
📍 จัดการข้อมูล

  2. 2Tab ระยะทาง — กำหนดระยะทางระหว่างคู่สถานที่
  3. 3Tab Backhaul — ระยะทางขากลับ (อาจต่างจากขาไป)
💡ข้อมูลทั้งหมดบันทึกใน localStorage — ใช้งานได้ offline ไม่ต้องการ internet

  2. 2คลิกชื่อ ต้นทาง/ปลายทาง ใน header เพื่อ highlight เส้นทางนั้น
  3. 3ราคาที่ highlight สีทอง = เส้นทางที่เลือก / สีม่วง = markup พิเศษ

  2. 2ตั้ง % markup ที่ต้องการบวกเพิ่มจากต้นทุน
  3. 3แก้ไขราคาแต่ละเส้นทางได้โดยตรงในตาราง

  2. 2กด บันทึก — ข้อมูลส่งไป Google Sheets อัตโนมัติ
  3. 3แก้ไขงานที่มีอยู่ — คลิกแถวนั้น → แก้ไขข้อมูล → บันทึก

ตั้งค่า GAS Sync — Google Apps Script
เชื่อม Fleet Dashboard กับ Google Sheets สำหรับ sync สถานะรถข้ามอุปกรณ์
🔧 ขั้นตอนติดตั้ง GAS (ครั้งแรก)
  1. 1เปิด Google Sheet ของ trip data → Extensions → Apps Script
  2. 2ลบ code เดิมทั้งหมด → วาง code จาก 📋 GAS Script ในหน้า Settings ของ Fleet Dashboard
  3. 3กด Deploy → New deployment → Web app
  4. 4ตั้งค่า: Execute as = Me / Who has access = Anyone → Deploy
  5. 5คัดลอก URL ที่ได้ → วางในช่อง GAS URL ใน Settings → กด 💾 บันทึก
  6. 6กด 🔌 Test — ถ้า ✅ แสดงว่าเชื่อมต่อสำเร็จ
ℹ️GAS sync ใช้สำหรับ sync สถานะรถ + Note ข้ามอุปกรณ์ และ list sheet tabs สำหรับ Statement วิเคราะห์
🔄 Re-deploy เมื่อ update code
  1. 1Copy code ใหม่จาก Settings (กด 📋 Copy Code)
  2. 2Apps Script → วาง code ใหม่ทับ → Deploy → Manage deployments
  3. 3กด Edit (ดินสอ) → Version = New version → Deploy
  4. 4URL เดิมยังใช้ได้ ไม่ต้องอัปเดตใน Settings

📁 เตรียม Google Sheet

  2. 2ตั้งชื่อ tab ตามเดือน รูปแบบ YYYY-MM เช่น 2026-04, 2026-05

  4. 4Share sheet → Anyone with the link → Viewer
  5. 5คัดลอก Sheet ID จาก URL (ส่วนระหว่าง /d/ และ /edit)

  1. 1Fleet Dashboard → ⚙️ ตั้งค่า GAS Sync → หัวข้อ 🏦 Transaction Report
  2. 2วาง Sheet ID → กด 💾 บันทึก
  3. 3ไปหน้า 🏦 Statement วิเคราะห์ → กด ☁️ ดึงจาก Sheets
