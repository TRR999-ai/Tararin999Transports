🏦 Transaction Report — Google Sheets

        

          เชื่อม Statement วิเคราะห์กับ Google Sheets แบบ tab ต่อเดือน


          ขั้นตอนการตั้งค่า:

          1. สร้าง Google Sheet ใหม่ (หรือใช้ไฟล์เดิม)

          2. ตั้งชื่อ tab ตามเดือน เช่น 2026-04 2026-05

          3. วาง data จาก TMB Transaction Report export ทับได้เลย (ไม่ต้องแก้ format)

          4. Share sheet → "Anyone with the link" → Viewer

          5. คัดลอก Sheet ID จาก URL แล้วใส่ด้านล่าง

          6. GAS Script ต้องอัปเดตแล้ว Re-deploy (ดูหน้า GAS Script ด้านบน)


          Sheet ID คือส่วนนี้ใน URL:

          docs.google.com/spreadsheets/d/← Sheet ID อยู่ตรงนี้ →/edit
